# Application de suivi de protocole microbiologique

Cette mini-application web résume un protocole de laboratoire et permet de suivre les étapes avec :

- un résumé des étapes,
- une checklist persistante (localStorage),
- une barre de progression,
- un minuteur pour l'étape sélectionnée.

## Lancer l'application

Ouvrir simplement `index.html` dans un navigateur.

Ou via un serveur local :

```bash
python3 -m http.server 8000
```

Puis visiter `http://localhost:8000`.
